# MiniTwit

A twitter-clone application built with Flask.

## Getting Started

### Initialize the database
```bash
./control.sh init
```

### Run the application
```bash
./control.sh start
```

Or with production server:
```bash
./control.sh startprod
```

### Stop the application
```bash
./control.sh stop
```

## Access

Once running, access the site at: **http://localhost:5000**
